동작하는 깔끔한 코드 (Clean code that works)라는 말을 좋아합니다. 비즈니스 복잡함을 잘 다루고 변경하기 쉬운 코드를 작성하기 위해 고민합니다. 이를 위해 객체지향, 도메인 주도설계, 단위 테스트에 관심이 많습니다.
